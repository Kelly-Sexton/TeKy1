TEKY Weekend Homework

Instructions

Answer each problem in a JavaScript file formatted like so:

// Name, TA Name, Date

// Problem 1 
Problem 1 Code Here

// Problem 2 
Problem 2 Code Here

// Problem 3
Problem 3 Code Here

// Problem 4 
Problem 4 Code Here

// Problem 5 
Problem 5 Code Here

// Problem 6
Problem 6 Code Here

Name the file yourname.js. Submit it to your TA through a private message on Slack. Good luck and happy coding!

Example Problem + Answer

Write a function to output a string to the console.

// Example Question
function printString(myString) {
    console.out(myString);
}

print("This is a string");

Problem 1

Show how you would determine the location of the number 27 in the array:

var numArray = [18, 24, 57, 98, 45 27, 3, 456, 3, 35, 85];

Store its location in a variable.

Problem 2

Using the same array from Problem 1, calculate the sum of all numbers in the array and assign the sum to a variable.

Problem 3

Initialize a new array and fill it with the numbers 1 through 10. Then, remove the last element of the array. Output the array to the console.

Problem 4

Create a function with two parameters. This function will be passed two numbers as arguments. Calculate the sum of the parameters. If the sum is less than or equal to 25, the function should return true. If not it should return false. Be sure to use an appropriate function name.

Problem 5

Create a function with two parameters. This function will be passed two strings as arguments. Combine both of these strings and assign the result to a variable. If this new string is longer than 12 characters, return "Error, string too long.". Otherwise, return the variable that stores the new string. Be sure to use an appropriate function name.

Problem 6

Design a while loop that counts from 0 to 20 using the variable num. If num is between 0 and 5 output "Very Low Number" to the console. If it is between 6 and 10 output "Low Number" to the console. If it is between 11-15 output "High Number" to the console. For everything else output "Very High Number" to the console.
